# Code Simplifier Skill

코드 정리 및 단순화 스킬

## 목적

Claude가 작업을 완료한 후 코드를 정리하고 단순화합니다.

## 적용 시점

- 새 기능 구현 완료 후
- 버그 수정 완료 후
- 리팩토링 완료 후

## 수행 작업

### 1. 불필요한 코드 제거
- 사용하지 않는 import 제거
- 사용하지 않는 변수 제거
- 주석 처리된 코드 제거
- console.log/debug 문 제거

### 2. 코드 정리
- 일관된 포맷팅 적용
- 함수/변수 네이밍 개선
- 중복 코드 통합

### 3. 타입 개선 (TypeScript)
- any 타입 구체화
- 타입 추론 활용
- 인터페이스/타입 정리

### 4. 성능 최적화
- 불필요한 리렌더링 방지 (React)
- 메모이제이션 적용 (useMemo, useCallback)
- 비효율적인 로직 개선

## 사용 방법

```
이 스킬을 호출하면 최근 수정된 파일들을 분석하고
위의 작업들을 자동으로 수행합니다.
```

## 적용 규칙

### 반드시 적용
- ESLint 규칙 준수
- Prettier 포맷팅 적용
- 타입 안전성 유지

### 적용하지 않음
- 기능 변경 (로직 수정)
- 새로운 기능 추가
- 기존 API 인터페이스 변경
