# Verify App Skill

Post Style 애플리케이션 E2E 검증 스킬

## 목적

배포 후 또는 주요 변경 후 애플리케이션의 핵심 기능을 자동으로 검증합니다.

## 검증 시나리오

### 1. 기본 접속 테스트
- [ ] http://localhost:3000 접속 가능
- [ ] 메인 페이지 로드 완료
- [ ] 에러 없음

### 2. 인증 테스트
- [ ] 로그인 페이지 접속
- [ ] 테스트 계정으로 로그인 (master@poststyle.com)
- [ ] 로그인 성공 확인
- [ ] 로그아웃 테스트

### 3. 핵심 기능 테스트

#### 콘텐츠 관리
- [ ] 콘텐츠 목록 페이지 접속
- [ ] 콘텐츠 생성 페이지 접속
- [ ] (선택) 콘텐츠 생성 테스트

#### 스타일 관리
- [ ] 스타일 목록 페이지 접속
- [ ] 스타일 학습 페이지 접속

#### AI 기능
- [ ] AI 콘텐츠 생성 페이지 접속
- [ ] SEO 분석 페이지 접속

### 4. 관리자 기능 (Admin 계정)
- [ ] 대시보드 접속
- [ ] 사용자 관리 페이지 접속
- [ ] 분석 페이지 접속

## 사용 방법

```
이 스킬을 호출하면 Chrome MCP 또는 Playwright MCP를 사용하여
위의 모든 시나리오를 순차적으로 검증합니다.
```
