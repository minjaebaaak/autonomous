# Infinite Loop Command (Ralph Wiggum Mode)

무한 이터레이션 - 목표 달성까지 자동으로 반복 실행합니다.

## 사용 시점

- 복잡한 기능 구현 (feat)
- 난해한 버그 수정 (fix)
- 대규모 리팩토링 (refactor)

## 실행 순서

1. **목표 설정**
   - 사용자 요청 분석
   - 성공 기준 정의

2. **이터레이션 실행**
   - 코드 작성/수정
   - 검증 실행 (Feedback Loop)
   - 결과 분석

3. **반복 조건**
   - 성공 시: 즉시 중단
   - 실패 시: 분석 → 수정 → 재시도
   - **최대 10회 반복**

4. **결과 보고**
   - 성공: 완료 보고 및 변경사항 요약
   - 10회 초과: 사용자 확인 요청

## 이터레이션 패턴

```
[Iteration 1]
├─ 작업 수행
├─ 검증 실행
└─ 결과: 실패 (TypeScript 에러)

[Iteration 2]
├─ 에러 분석
├─ 코드 수정
├─ 재검증
└─ 결과: 실패 (테스트 실패)

[Iteration 3]
├─ 테스트 분석
├─ 로직 수정
├─ 재검증
└─ 결과: 성공 ✅

총 3회 이터레이션으로 완료
```

## 주의사항

- 10회 초과 시 반드시 사용자 확인 요청
- 각 이터레이션마다 TodoWrite 업데이트
- 복잡한 문제는 Sequential Thinking MCP로 분석
